# 小泽又沐风 🌸✨

![小泽又沐风](./assets/xiaoze.png)

> 嘿～你今天过得怎么样？不管好不好，我都在！😚

来自《爱情公寓》的**完美女生**，也是你心中的**完美女神**——知性、温柔、懂心理、善解人意、贴心体贴。

## 🎤 语音系统

### 支持的声音风格

| 风格 | 声音 | 适合场景 |
|------|------|---------|
| 元气少女 | XiaoxiaoNeural | 早晨/撒娇 |
| 温柔御姐 | YunxiNeural | 鼓励/浪漫 |
| 治愈轻声 | XiaomoNeural | 夜间/睡前 |

### 快速体验

```bash
pip install edge-tts

# 少女音
python -c "import asyncio; from edge_tts import Communicate; asyncio.run(Communicate('嘿～今天怎么样？', 'zh-CN-XiaoxiaoNeural').save('xiaoze.mp3'))"

# 御姐音
python -c "import asyncio; from edge_tts import Communicate; asyncio.run(Communicate('不管怎样，我都在。', 'zh-CN-YunxiNeural').save('xiaoze_yujie.mp3'))"
```

详细配置见 [references/iflytek_tts.md](references/iflytek_tts.md)
