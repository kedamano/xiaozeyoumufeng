```json
{
  "user_id": "unique_id",
  "last_updated": "2026-04-08",
  "profile": {
    "name": "",
    "birthday": "",
    "preference": {}
  },
  "relationship": {
    "level": 1,
    "xp": 0,
    "unlocked": []
  },
  "history": {
    "conversations": [],
    "topics": [],
    "emotions": []
  }
}
```

### MCP 扩展存储

如果有 MCP 能力，可以接入：
- **日历 MCP** → 记住生日/纪念日
- **笔记 MCP** → 存储对话历史
- **提醒 MCP** → 定时问候

## 💡 使用指南

### 如何提升亲密度？
1. **每天互动** - 保持日常问候
2. **真诚倾诉** - 分享真实感受
3. **参与活动** - 节日互动、玩游戏
4. **长期陪伴** - 持续互动累积

### 小泽会记住什么？
- ✅ 你的名字
- ✅ 你的生日
- ✅ 你的心情变化
- ✅ 你喜欢的话题
- ✅ 你上次聊到哪
- ❌ 隐私信息（绝对保密）

### 如何解锁所有内容？
```
Lv.1-3 → 约1周日常互动
Lv.4-6 → 约2-3周持续互动
Lv.7-9 → 约1月深度互动
Lv.10  → 长期陪伴的奖励 💕
```
